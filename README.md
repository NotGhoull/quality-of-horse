# Quality of Horse
[![Latest Release](https://gitlab.com/NotGhoull/quality-of-horse/-/badges/release.svg)](https://gitlab.com/NotGhoull/quality-of-horse/-/releases) [![CurseForge download](https://img.shields.io/curseforge/dt/1500027?style=flat&label=CurseForge%20downloads)](https://www.curseforge.com/minecraft/mc-mods/quality-of-horse) [![Modrinth download](https://img.shields.io/modrinth/dt/RVR0jj4y?style=flat&label=Modrinth%20downloads)](https://modrinth.com/mod/quality-of-horse)    
**Quality of Horse** (qoh) is a light-weight and highly configurable vanilla+ mod that makes horses more useful.

> [!TIP]
> For more in depth details, please make sure you read the [Wiki](https://gitlab.com/NotGhoull/quality-of-horse/-/wikis/)

## Features
 - Two-player riding
 - Horse inventories
 - Horses follow you
 - Improved mobility  
 - & More!

## Installation
#### Requirements
 - Architectury API
 - YetAnotherConfigLib (YACL)
 - Fabric API (Fabric only)

#### Supported platforms
 - Fabric (1.21+)
 - NeoForge (1.21+)
 - Forge is **not supported**, but forks are welcome. Eventually, we hope to support forge.

#### Downloads
 - [CurseForge](https://www.curseforge.com/minecraft/mc-mods/quality-of-horse)
 - [Modrinth](https://modrinth.com/mod/quality-of-horse)

## Configuration
QoH includes full configuration support
 - (General) Config file: `config/qoh.json5`
 - (Fabric) Mod menu integration if you use [Mod Menu](https://modrinth.com/mod/modmenu)
 - (NeoForge) Config is all built-in to the mod's config screen

## Usage
All mechanics (like chests, riding, and behaviors) are explained in detail in the [Wiki](https://gitlab.com/NotGhoull/quality-of-horse/-/wikis/Mechanics).
